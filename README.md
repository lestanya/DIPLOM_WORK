--------------------------------------------------------------------


⚙️ Технические требования




Операционная система: Windows версии 10/11 на 64 бит.


Язык программирования: Python 3.12


Среда разработки: VisualStudioCode (рекомендуемый вариант), либо Pycharm



📌Прочее: права администратора для выполнения bat скриптов



--------------------------------------------------------------------



🔌 Запуск



1. Скачайте проект с гитхаба с помощью следующих интсрументов:
   
   - HTTPS: https://github.com/lestanya/DIPLOM_WORK.git
   - Download ZIP
     
3. Откройте терминал при помощи сочетания клавиш Ctrl + P, либо на вкладке сверху под названием Terminal ➡️ New Terminal
4. введите названия файлов по порядку

       start.bat
   
   _📖 этот файл создает виртуальное окружение и автоматически проверяет наличие всех зависимостей, необходимых для запуска данного проекта_


   ---------------------------------------------------------------------

       manage.bat
   
   _📖 в данном файле находится удобная панель для совместного управления как чат-ботом, так и административной панелью_

🗺️ Общая информация о структуре и функционале проекта:


 -------------------------------------------------------------

  -- /app 
  
    _📖 главное приложение, которое управляет чат-ботом и взаимодействует с другими модулями
    

  -- /flask_app 
  
    _📖 административная панель для просмотра заявок и статистики


  -- /data 
  
    _📖 папка с весами моделей, энкодерами, токенизатором, а также справочными материалами, необходимых для модуля обработки текста
