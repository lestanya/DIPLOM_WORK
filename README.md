_____________Требования_____________________

Операционная система: Windows версии 10/11 на 64 бит.
Язык программирования: Python 3.12
Среда разработки: VisualStudioCode (рекомендуемый вариант), либо Pycharm

Прочее: права администратора для выполнения bat скриптов

---------------------------------------------

_____________Запуск_____________________

1. Скачайте проект с гитхаба с помощью [].
2. Откройте терминал при помощи сочетания клавиш Ctrl + P, либо на вкладке сверху под названием Terminal ➡️ New Terminal
3. введите названия файлов по порядку

-- start.bat 📖 этот файл создает виртуальное окружение и автоматически проверяет наличие всех зависимостей, необходимых для запуска данного проекта.

-- manage.bat 📖 в данном файле находится удобная панель для совместного управления как чат-ботом, так и административной панелью.
